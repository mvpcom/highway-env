Parameters:

  • surface – the pygame surface on which the node is displayed
  • origin – the location of the node on the surface [px]
  • size – the size of the node on the surface [px]
  • temperature – the temperature used for exploration bonus visualization
  • depth – the depth of the node in the tree
  • selected – whether the node is within a selected branch of the tree
  • display_exploration – display the exploration bonus
  • display_count – display the visitation count of the node
  • display_text – display a text showing the value and visitation count of the node
  • display_prior – show the prior probability of each action
